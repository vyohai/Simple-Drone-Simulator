This method will return the control command(rottors_speeds) based on the current and previous states, each state contains: quaternion, position, velocity, angular velocity, for now only the altitude control is relevant.

Parameters
referance(arma::vec3)- wanted location.
current_step(std::array<float,13>)- body angular rates.
previous_step(std::array<float,13>)- the rottors speed command.
controllers_coefficients(std::array<std::array<float,3>,6>)- (kp,ki,kd) triplets for all 6 PID's.
Integral_altitude(float*) - the integral of the altitude error.
Returns
arma::vec4, control output(rottors_speeds).
